# Wifi analyzer

A program for testing the wifi-connection.

## Setup

**Step 1: .env**

Correct values in `.env`

**Step 2: Venv**

Read the "Venv"-section of the Makefile.

## Useful commands

**Trace the dig**

```
dig +trace google.com
```

**What to look for**

- Is it resolving the IP in the router? 

**Ping and traceroute**

```
ping 8.8.8.8
traceroute 8.8.8.8
```

**What to look for**

Holes in the cheese.
