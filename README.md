# Wifi tester

A program for testing the wifi-connection.

## Setup

Do this to set it up:

```
py -m venv wifianalyzerenv
source wifianalyzerenv/bin/activate
```

## Useful commands

**Trace the dig**

```
dig +trace google.com
```

**What to look for**

- Is it resolving the IP in the router? 

**Ping and traceroute**

```
ping 8.8.8.8
traceroute 8.8.8.8
```

**What to look for**

Holes in the cheese.
